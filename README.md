# bbq-arduino-ionic
An Ionic 2 app that communicates with an Arduino board via MQTT, to monitor the temperature on a charcoal grill and adjust with fans.

# Ionic 2.x
## Installation
1. Install Node >= 6.x
1. Install Cordova & Ionic CLI
    * `npm install -g cordova ionic`
1. Run: `npm install`
1. Run: `ionic serve`
